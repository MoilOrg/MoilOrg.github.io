Reverse View¶

Overvivew

This example demonstrated how to generate image with new center optical point.

Variable

  ο image = The original fisheye image
  ο alpha_max = max of alpha. The recommended value is half of camera FOV
  ο alpha = the value of alpha
  ο beta = the value of alpha

Example Code

1 Reverse view

$ reverse = moildev. panorama (image, alpha_max, alpha, beta)

Purpose:

This function was used for transforming the fisheye image to change the optical point without manipulating the camera head.

Parameter::

  ο image = The original fisheye image
  ο alpha_max = max of alpha. The recommended value is half of camera FOV.
  ο beta = the value of alpha.
  ο See example

2. Code Overview

First, import and create object of moildev library

# import library
 import cv2 from Moildev
import Moildev

# create object of Moildev
moildev = Moildev ("../Raspi_Cam.json") # select parameter file (.json)

Create specified variable

create specified variable value of alpha beta zoom and mode
 alpha = 110
 alpha = 30
 beta = 30

Read image from directory

# read image from directory using opencv
image = image. cv2.imread ("../image.jpg")

Create reverse image

# create reverse image
reverse_image = moildev. reverseImage (image, alpha_max, alpha, beta)

# fill the variable
anypoint_image_1 = cv2.remap (image, map_x, map_y, cv2.INTER_CUBIC)

3. Save image using opencv cv2.imwrite

# save reverse image
 cv2.imwrite ("reverse_image.jpg", reverse_image)
