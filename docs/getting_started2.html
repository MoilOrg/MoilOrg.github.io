        Mulai¶

           Moildev Library adalah kumpulan fungsi yang mendukung pengembangan aplikasi gambar mata ikan berbasis bahasa pemrograman python. Anda dapat merujuk ke tautan ini untuk mendalami perpustakaan ini. saat ini perpustakaan dapat mendukung untuk:

        Perpustakaan apa yang digunkan dalam aplikasi ini

           Pada sesi ini, kita akan membahas sedikit tentang library yang digunakan. ini bisa menjadi referensi anda dalam mengembangkan plugin untuk aplikasi yang ingin anda buat. Mari lihat:

        a. Pyqt5

           PyQt adalah pustaka yang memungkinkan untuk menggunakan kerangka kerja Qt dari Python. Qt adalah aplikasi lintas platform gratis dan sumber terbuka yang ditulis dalam C ++ dan merupakan widget toolkit yang membuat GUI. Qt menawarkan solusi pemrograman dengan konsep pemrograman berorientasi objek (OOP) yang kuat: di mana masalah pemrograman diselesaikan dalam instance objek dengan atribut, fungsi, dan interaksi antar objek. Qt memiliki alat yang menyederhanakan pengembangan program dan antarmuka grafis pada desktop, sistem tertanam, dan perangkat seluler, seperti Qt Framework, IDE pembuat Qt, dan rantai alat. Menggunakan pustaka Qt dari Python, memungkinkan untuk membangun aplikasi lebih cepat tanpa mengorbankan banyak kecepatan di C++. Untuk instalasi perpustakaan ini, Anda dapat merujuk ke mengikuti tautan ini.

        b. OpenCV-python

            OpenCV standar untuk Perpustakaan Visi Komputer Sumber Terbuka. Ini adalah perpustakaan sumber terbuka dan gratis untuk pembelajaran mesin dan visi komputer. Di mana diluncurkan secara konvensional pada tahun 1999, dikembangkan oleh Gary Brodsky di Intel yang ditulis dalam bahasa pemrograman C/C++. Saat ini, kode sumber OpenCV dan modul tambahan yang memanggil kontribusi OpenCV tersedia di GitHub. Oleh karena itu kode mudah dimodifikasi untuk kebutuhan tertentu. OpenCV tersedia untuk berbagai bahasa pemrograman seperti C++, C, Java, MATLAB, dan Python serta mendukung sistem operasi seperti Linux, Microsoft Windows, Mac OS, dan Android. Karena memanfaatkan instruksi MMX dan SSE saat tersedia bersama antarmuka CUDA dan OpenCL, sebagian besar aplikasi visi waktu nyata dimungkinkan. Perusahaan mapan seperti Microsoft, Intel, Google, IBM, Yahoo, Sony, Honda, dan Toyota menggunakan perpustakaan ini.

        c. Moildev

           Perpustakaan moildev adalah kumpulan fungsi yang digunakan untuk mengembangkan aplikasi gambar mata ikan. Perpustakaan ini dikembangkan oleh The Omnidirectional Imaging and Surveillance Laboratory (OIL) di Ming Chi University of Technology, Taiwan, di bawah bimbingan Profesor. Chuang-Jan Chang. Awalnya perpustakaan ini menulis dalam C++ untuk memanfaatkan kecepatan komputasi yang disediakan oleh bahasa pemrograman ini. Saat ini, perpustakaan ini tersedia untuk bahasa pemrograman python, yang menggunakan antarmuka pemrograman aplikasi (API) yang mengikat yang menyediakan kode perekat yang dibuat khusus untuk memungkinkan bahasa pemrograman menggunakan perpustakaan asing atau layanan sistem operasi dengan kecepatan yang sama. Pustaka Moildev kompatibel dengan sistem operasi (OS) Linux dan Windows. Library Moildev memiliki fungsi, seperti Anypoint mode-1, Anypoint mode-2, dan Panorama. Setiap fungsi memiliki parameter dan kontrolnya sendiri dengan memberikan sudut zenithal dan sudut azimut untuk mencapai wilayah tertentu yang diinginkan. Sebelum menerapkan fungsi Moildev, konfigurasi kamera-init diperlukan. Ini memberikan properti kamera parameter intrinsik dan ekstrinsik dari kalibrasi kamera.

        d. Perpustakaan lainnya

           Pustaka lain perlu digunakan untuk melengkapi aplikasi tergantung pada desain dan apa yang akan digunakan di masa mendatang. PyPI menyediakan lebih dari seratus ribu perpustakaan yang dapat digunakan untuk mencapai tujuan kami.

        Instalasi

        1. Bagaimana cara meng-install¶

        Pustaka ini sekarang tersedia dalam distribusi pypi disini, untuk menginstal menggunakan distribusi python package index (PyPI), Anda dapat menggunakan perintah di bawah ini:

        $ pip3 install moildev

        Atau Anda dapat menginstal secara manual dengan mengkloning repositori ini ke direktori lokal Anda, lalu ikuti perintah ini:

        $ cd Moildev
        $ pip install -r requirments.txt
        $ pip install Moildev

        2.1 Membuat Objek

        from Moildev import Moildev

        Setiap kamera fisheye dapat dikalibrasi dan mendapatkan serangkaian parameter oleh laboratorium MOIL, sebelum fungsi yang berurutan dapat bekerja dengan benar, konfigurasi diperlukan di awal program.

        Untuk membuat objek dari Moildev, Anda harus meneruskan parameter kamera ke paket. Ada 3 cara untuk membuat objek moildev:

        1. Buat objek menggunakan argumen kata kunci input

        2. Buat objek menggunakan satu parameter dengan ekstensi format file json

        3. Buat objek menggunakan beberapa parameter dengan ekstensi format file json
