        MoilApp¶

            MoilApp adalah perangkat lunak pengolah gambar mata ikan yang dikembangkan oleh MOIL-Lab. Aplikasi ini merupakan implementasi dari Moildev SDK sebagai metode canggih dalam pengolahan citra mata ikan. Yang umum digunakan adalah mengolah citra fisheye dengan hasil tampilan panorama dan tampilan Anypoint. Metodologi canggih untuk pemrosesan gambar ini sangat berguna dalam berbagai aplikasi yang membutuhkan kualitas gambar tinggi dan FoV sudut lebar. Aplikasi ini adalah plugin pendukung untuk aplikasi tertentu seperti pengukuran 3D, multiple view untuk ADAS, axis controller dan lain-lain. Plugin adalah komponen perangkat lunak yang menambahkan fitur tertentu ke program komputer yang ada. Ketika sebuah program mendukung plug-in, itu memungkinkan penyesuaian.

        Ada banyak manfaat untuk membangun Aplikasi dengan kerangka kerja plugin:

          - Pengembang pihak ketiga dapat membuat dan memperluas Aplikasi Anda
          - Fitur baru lebih mudah dikembangkan
          - Aplikasi Anda menjadi lebih kecil dan lebih mudah dipahami

        Ini adalah versi terbaru dari MoilApp dimana sebelumnya kami memiliki versi 1, versi 2 dan versi 3.

          - FoV pada parameter kamera
          - Pilih jenis kamera jika tidak tersedia pada gambar metadata
          - Integrasikan dengan Moildev versi 2.6.0 yang dapat memproses untuk gambar fisheye sumbu optik terbaru
          - Atur Ulang Gambar Titik Pusat

        Seting MoilApp didalam Komputer Anda¶

           Saat Anda ingin menjalankan MoilApp di sistem Anda, ada beberapa persyaratan yang diperlukan seperti Ubuntu versi 18 atau lebih tinggi dengan terinstal python versi >3.6 dan IDE. Untuk memudahkan dalam pengembangan, kami menggunakan IDE untuk menulis program kami. Integrated development environment (IDE) adalah aplikasi perangkat lunak yang menyediakan fasilitas lengkap bagi pemrogram komputer untuk pengembangan perangkat lunak. IDE biasanya terdiri dari setidaknya editor kode sumber, alat otomatisasi build, dan debugger. PyCharm adalah IDE yang digunakan dalam pemrograman komputer, khusus untuk Python. Anda dapat dengan mudah menginstal mengikuti panduan yang diberikan di situs web resmi di sini. Oke mari kita mulai setup MoilApp:

        Note: The project was testing on Ubuntu 18.04 and 20.04 LTS

        Bagaimana menjalankan aplikasi ini

        Ikuti langkah ini sebelum Anda menggunakan MoilApp;

        Buka Terminal Anda dan klon repositori ini

        $ git clone https://github.com/MoilOrg/MoilApp.git

        Ubah Direktori ke folder MoilApp

        $ cd MoilApp/

        Instal semua persyaratan menggunakan skrip bash yang kami sediakan dengan menggunakan Perintah ini

        $ . build.sh

        Menjalankan proyek ini

        $ cd src/
        $ python main.py

        Pengoprasian MoilApp¶

           Pada sesi ini kami akan menjelaskan komponen-komponen dalam aplikasi MoilApp dan tutorial cara menggunakan aplikasi tersebut.

        Antarmuka Pengguna


        Berikut penjelasan detailnya:

        1. Menu bar
          This container stores various button
          a. File
           File berisi Load image, Load video, Open cam, Parameter Cam, Record Video, Save Image, dan Exit
          b. Window
           Window berisi Maximize dan Minimize
          c. Apps
           Aplikasi berisi Tambah, Buka, Hapus, Buat dan Bantuan untuk Pengaya
          d. Help
           Bantuan berisi Tentang Aplikasi, Tentang Kami, dan Aksesibilitas
        2. Title application
          Bagian ini akan memberikan informasi tentang nama aplikasi dan informasi tentang lokasi gambar dan video sebagai sumber.
        3. Source media button
          grub ini berisi tiga tombol: memuat gambar, memuat video, dan membuka kamera. Anda dapat memuat gambar dan video dari direktori Anda dan juga bisa mendapatkan gambar dari kamera Anda dengan streaming video. Pada kamera terbuka Anda dapat menggunakan Kamera USB dan kamera streaming.
        4. Rotate button
          putar Memutar gambar ke arah searah jarum jam dan berlawanan arah jarum jam.
        5. Zoom button
          Membuat gambar terlihat lebih besar atau lebih kecil dengan memperbesar dan memperkecil.
        6. Saving button
          simpan Simpan gambar yang dihasilkan atau video rekaman di direktori lokal Anda
        7. Plugin controller
          Container Plugin controller berisi beberapa tombol yang berfungsi untuk mengontrol plugin seperti menambah, membuka dan menghapus aplikasi plugin dari aplikasi utama.
        8. Information label
          Menampilkan informasi jenis kamera yang digunakan.
        9. Selection mode view
            Wadah ini berisi beberapa tombol pemilihan mode tampilan termasuk tampilan asli, tampilan titik mana pun, dan tampilan panorama.
        10. Help button
          Berisi beberapa tombol yang menampilkan informasi dari aplikasi ini seperti panduan manual, tentang kami dan tentang MoilApp
        11. Clear media from widget
          Tombol ini akan menghapus semua objek seperti gambar pixmap dan lainnya dari antarmuka pengguna agar aplikasi terlihat seperti baru dibuka
        12. Extra button
          ini akan aktif saat Anda memilih Anypoint atau tampilan panorama dan membantu Anda menjelajahi gambar.
        13. Widget show result image
          Widget ini akan menampilkan gambar hasil setelah diproses, ini adalah penampil gambar utama pada aplikasi antarmuka pengguna.
        14. Widget to show Image saved
          Menampilkan gambar yang berhasil disimpan yang kemudian dapat dibuka kembali jika media sumber Anda adalah video atau kamera.
        15. Video controller
          controller Wadah video controller berisi beberapa tombol dan label yang berfungsi untuk mengontrol media seperti start, pause, stop, forward, backward, time slider dan label durasi.
        16. Widget show original image
          Widget ini berfungsi untuk menampilkan dan memelihara gambar asli untuk memberikan referensi mengenai proses yang terjadi.
