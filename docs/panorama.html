Panorama¶

Overview

This example demonstrated how to generate panorama image centered at the 3D direction with alpha = iC_alpha_degree and beta = iC_beta_degree.

Moildev provide two option to create panorama image. you only need to choose one:

1. getPanoramaMaps()

This function will give you return maps x and maps y for panorama image. after get maps x and maps y we can create panorama image using remap function from opencv.

2. panorama()

This function will give return a panorama image directly.

Variable
- image = the original image
- alpha_min = the minimum alpha
- alpha_max : max of alpha. The recommended value is half of camera FOV.

Example Code

1. Create panorama maps

$ map_x, map_y = moildev. getPanoramaMaps (alpha_min, alpha_max)

Purpose:

To generate a pair of X-Y Maps for alpha within 0..alpha_max degree, the result X-Y Maps can be used later to generate a panorama image from the original fish-eye image.

Parameter::

- alpha_min = the minimum alpha
- alpha_max = the maximum of alpha. The recommended vaule is half of camera FOV.
- For example, use 90 for a 180 degree fisheye images and use 110 for a 220 degree fisheye images.
- See example

2. Panorama

$ panorama_image = moildev. panorama (image, alpha_min, alpha_max)

Purpose:

The panorama image centered at the 3D direction with alpha = iC_alpha_degree and beta = iC_beta_degree.

Parameter::

- image = the original image
- alpha_min = the minimum alpha
- alpha_max : max of alpha. The recommended vaule is half of camera FOV. For example, use 90 for a 180 degree fisheye images and use 110 for a 220 degree fisheye images.
- See example

3. Code Overview

First, import and create object of moildev library

# import library
 import cv2 from Moildev
import Moildev

moildev = Moildev ("../Raspi_Cam.json") # select parameter file (.json)

Create specified variable value of alpha, beta, zoom and mode

# create specified variable value of alpha min dan alpha max
 alpha_max = 110
 alpha_min = 110

Read image from directory

# read image from directory using opencv
 image = cv2.imread ("../image_2.png")

You only need choose one between this two function

1. Create maps and panorama image using remap function After get maps x and maps y, next add function cv2.remap from opencv

Read image from directory

# create maps_x and map_y image
 map_x, map_y = moildev. getPanoramaMaps (image, map_x, map_y, cv2.INTER_CUBIC)

panorama_image_1 = cv2.remap

2. Create panorama image directly image

# create panorama image
 panorama_image = moildev. panorama (image, alpha_min, alpha_max)

3. Save image using opencv imwrite

# save panorama image
 cv2.imwrite ("panorama_image_1.jpg", panorama_image_1)
 cv2.imwrite ("panorama_image_2.jpg", panorama_image_2)
