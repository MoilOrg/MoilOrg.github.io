        Anypoint¶

        Overview

            Contoh ini menunjukkan cara menghasilkan gambar tanpa distorsi atau gambar tampilan titik mana pun menggunakan fungsi getAnypointMaps() dan anypoint(). Untuk mode 1, rotasi yang dihasilkan adalah rotasi derajat betaOffset di sekitar sumbu Z (roll) setelah rotasi derajat alfaOffset di sekitar sumbu X (pitch). untuk mode 2, Rotasi hasil adalah rotasi derajat theta_Y di sekitar sumbu Y (yaw) setelah rotasi derajat theta_X di sekitar sumbu X (pitch).

        Moildev menyediakan dua opsi untuk membuat gambar titik apa pun, Anda hanya perlu memilih satu:
        1. getAnypointMaps()
            Fungsi ini akan membuat Anda mengembalikan peta x dan peta y untuk angel zenital dan azimutal yang ditentukan. kembalinya XY Maps dapat digunakan nanti untuk memetakan kembali gambar asli menjadi gambar yang tidak terdistorsi. secara default fungsi ini akan memberi Anda mode-1 titik apa pun. Anda dapat menggunakan opencv remap untuk membuat gambar titik mana pun.

        2. anypoint()
           Fungsi ini akan memberikan gambar tanpa distorsi atau gambar titik mana pun secara langsung tanpa menggunakan fungsi remap lagi.
        Variable
          - gambar: gambar sumber
          - alfa: nilai alfa
          - beta: nilai beta
          - zoom: faktor zoom desimal, biasanya 1 hingga 12
          - mode: mode tampilan titik mana pun (secara default akan menjadi mode 1)

        Contoh Kode

        1. Buat peta titik mana pun

        $ map_x, map_y = moildev. getAnypointMaps getAnypointMaps(alpha, beta, zoom, mode=1)

        Tujuan::

        Tujuannya adalah untuk menghasilkan sepasang XY Maps untuk parameter alpha, beta dan zoom yang ditentukan, hasil XY Maps nantinya dapat digunakan untuk memetakan kembali gambar mata ikan asli ke gambar sudut target.

        Parameter::

          - alpha: alpha
          - beta: beta
          - zoom: decimal zoom factor, normally 1..12
          - selection anypoint mode(1 or 2), by default if will be mode 1
          - See example

        2. Anypoint

        $ anypoint_image = moildev. anypoint anypoint(image, alpha, beta, zoom, mode=1)

        Tujuan:

        Hasilkan gambar tampilan titik mana pun. Untuk mode 1, rotasi yang dihasilkan adalah rotasi derajat betaOffset di sekitar sumbu Z (roll) setelah rotasi derajat alfaOffset di sekitar sumbu X (pitch). untuk mode 2, Rotasi hasil adalah rotasi derajat theta_Y di sekitar sumbu Y (yaw) setelah rotasi derajat theta_X di sekitar sumbu X (pitch).

        Parameter::

          - gambar: gambar sumber
          - alfa: alfa
          - beta: beta
          - zoom: zoom
          - mode: mode tampilan titik mana pun (secara default akan menjadi mode 1)
          - lihat example

        3. Gambaran Kode

        Pertama, impor dan buat objek perpustakaan moildev

        # import library
         import Moildev from Moildev
        import Moildev

        moildev = Moildev ("../Raspi_Cam.json") # select parameter file (.json)

        Buat nilai variabel tertentu dari alpha, beta, zoom dan mode

        alpha = 30
        beta = 30
        zoom = 4
        mode = 2


        Baca gambar dari direktori

        # read image from directory using opencv
        image = image. cv2.imread ("../image.jpg")

        Anda hanya perlu memilih satu di antara dua fungsi ini

        1. Buat gambar titik apa pun menggunakan fungsi getAnypointMaps dan remap Setelah mendapatkan peta x dan peta y, selanjutnya tambahkan fungsi cv2.remap dari opencv

        # create maps_x and map_y image
        map_x, map_y = moildev. getAnypointMaps getAnypointMaps(alpha, beta, zoom, mode)

        # create anypoint image using remap function from map x and map y
        anypoint_image_1 = cv2.remap (image, map_x, map_y, cv2.INTER_CUBIC)

        2. Buat titik mana pun menggunakan gambar fungsi titik mana pun Secara langsung membuat gambar titik mana pun dalam satu kode baris

        # create anypoint image
         anypoint_image = moildev. anypoint(image, alpha, beta, zoom, mode)

        3. Simpan gambar menggunakan opencv cv2.imwrite

        # save anypoint image
         cv2.imwrite ("anypoint_image_1.jpg", anypoint_image_1)
         cv2.imwrite ("anypoint_image_2.jpg", anypoint_image_2)
