List Example Code with Moildev¶

No Name Description Overview Details
 1. Get image width & height Return the value of alpha beta from the coordinate image given. Variable getImageWidth_Height
 2. Get Image Center X & Y Return the value of alpha beta from the coordinate image given. Variable (Icx = Image center in X-axis, Icy = Image center in Y-axis). getIcx_getIcy
 3. Get alpha beta Return the value of alpha beta from the coordinate image given. Variable (1. delta_x = pixel position of the image in x-axis, 2. delta_y = pixel position of the image in y-axis, 3. alpha = for create anypoint image 4. rho = degree). getAlpha_Beta_rho
 4. Recenter Image Return the image with change optical points. Variable (1. image = The original fisheye image, 2. alpha_max = max of alpha. The recommended value is half of camera FOV, 3. alpha = the value of alpha, 4. beta = the value of alpha). reverse_view
 5. Create Object Create object to be able to use all functions. Had three deference way to create object. We just need to choose one (1. Create object using input keyword arguments, 2. Create object using one parameter with json file format extension, 3. Create object using multiple parameter with json file format extension). object_image
 6. Anypoint Image Return image with Anypoint view (mode 1 and mode 2). Provide two option to create anypoint image, you only need to choose one: (1. getAnypointMaps(), 2. anypoint()) anypoint_image
 7. Panorama Image Return the panorama image result. Moildev provide two option to create panorama image. you only need to choose one: (1. getPanoramaMaps(), 2. panorama()) panorama_image
