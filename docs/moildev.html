What is Moildev?¬∂

    Moildev Library is a collection of functions to developments fisheye image applications. Initially, this library is written for C++ programing language. In the recent decade, python is a programming language that is very popular for developers. That is the reason why we are developing the Moildev library for python. This library build using share object library C++ to make it fast in computation and using ctypes to wrapper the function inside the library that can use in python. This library was available for Linux and Windows with installation python 64 bit.
