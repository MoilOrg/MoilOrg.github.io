    Example of Moildev library MoilApp is a fisheye image processing software developed by MOIL-Lab. This application is an implementation of Moildev SDK as a sophisticated method of processing fisheye images. The commonly used is to process fisheye image with the result panorama view and Anypoint view. This sophisticated methodology for image processing is very useful in a variety of applications that require high image quality and wide-angle FoV.
